Sistema Simples de Cadastro de Usuários em Python

Este é um projeto simples em Python que simula um sistema de cadastro de usuários no terminal.
O programa permite registrar múltiplos usuários armazenando nome, e-mail e senha em uma lista de dicionários.
Este projeto foi criado com fins de aprendizado e prática de lógica de programação em Python.

Simple User Registration System in Python
This is a simple Python project that simulates a user registration system in the terminal.
The program allows you to register multiple users by storing their name, email, and password in a dictionary list.
This project was created for learning and practicing programming logic in Python.
